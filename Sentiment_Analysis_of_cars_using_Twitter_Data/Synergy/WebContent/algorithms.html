NAIVE BAYESIAN ALGORITHM:

Naive Bayes classifier makes the assumption known as class conditional independence which can be explained as the values of the variable in a given class which is independent of the values of other variable. The major advantage of using Naive Bayes algorithm is that it requires small amount of training data to calculate the variable values necessary for predicting the outcome. The different combination of values of chosen variables are given to the classifier, the classifier return the posterior probability distribution for every class variable.
Bayes theorem of probability can be defined as the posterior probability in which there is a training pattern with attribute A which belongs to class Ck.







LEXICON BASED APPROACH:

Lexicon based approach is used to predict the sentiments of every individual texts or phrases. It contains a set of positive, negative words or phrases. Data which has been extracted from the twitter may contain the combination of positive and negative words or phrases. Theses tweets are preprocessed by applying normalizing functions, detection of hash tags , emoticon detection. Depending on the type of input data we can find the level of difficulty pf performing sentiment analysis. There are three important steps in Lexicon based approach for mining the sentiment of data:
1. Creation of lexicon
2. Preprocessing
3. Sentiment calculation



SENTIMENT ANALYSIS:

Sentiments of the users are being recognized with the help of hash tags, emoticons, positive and negative words. These sentiments are being calculated based on the values of the words assigned. The sentiments analysis can be done by using Breen's approach:

BREEN'S APPROACH:

The first step is to calculate the score for each tweet based on the positive and negative tweets.
Scores= Number of positive words - Number of negative words
If the score is greater than 0, it implies that the tweet is positive i.e, user has tweeted positive about that particular topic.
If the score is less than 0, it implies that the tweet is negative i.e, user has tweeted negative about that particular topic.
If the score is equals to 0, it implies that the tweet is 'nonpartisan presumption i.e, user has tweeted neither positive nor negative about that particular topic.
